Conscious

Overview

Conscious is an iOS and Android native application that encourages users to implement change by becoming a more conscious shopper. The application is intended to educate users on how the products and companies they choose, support or align with their values.

I was the UI/UX Designer for this project. I did everything from the initial concept and visual identity to user flows, wireframes, prototyping and the final screens and mockups. I worked on the project from early April to early June 2021.

My Process

Research

After reviewing a few different native applications that I found similar to my concept, I discovered features that would be beneficial to include. During this process I also noticed features that I wanted to improve upon based on competitor applications.

User Flow

During the UX Portion of this project I created a user flow diagram based on three user scenarios. The first “I want to browse companies and products that fit my values.” The second “I want to learn more about the companies I support and if they align to my values.” The last “I want to scan products I use.” All three of these were necessary to build the Minimum (or Most) Viable Product (MVP).

Wireframes

Using the latest Human Interface Guidelines and Material Designs I created two mid fidelity wireframes of the application that use standard UI elements from Apple and Android mobile operating system.

User Testing

I tested my mid fidelity wireframes on seven users. Many expressed their need for an application of this type, and praised the visual design. Each provided valuable feedback on some pain points in the application.

Final Design & Mockups

Through the helpful information I recived from my user testing I finalized my designs. I created eleven screens for each operating system, iOS and Android. All screens can be broken down into a few main categories depending on function and content. The first category is The Onboarding Process. These screens provide users with a short introduction of the application and the option to sign up or sign in. The second category is the Search and Filtering screens which take up a large portion of the conscious app as there are multiple screens that allow users to search. The last category is The Company And Product Infomation screens. These screens allow users to better understand in great detail the values a company or product might have in contrast to their own.

Prototype & Case Study

I created an interactive prototype that allows users to see each screen in action. View the iOS or the Android prototype. Also feel free to view the full case study on Behance!

Conclusion

My main goal was to adhere to the strict Human Interface and Material Guidelines to create an application that helps users become more conscious shoppers. Through research, wireframes, user feedback and multiple design iterations I was able to reach my goal, stay within each operating systems guidelines, and watch my application come to life.
